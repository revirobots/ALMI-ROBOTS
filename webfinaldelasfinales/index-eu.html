  // Kode nagusia

Kode Berritzailea

Programazio teknika aurreratuak erabili genituen, klasean ikusi ez direnak, gure roboten errendimendua eta gaitasunak hobetzeko.

// PID inplementazioa mugimendu kontrolerako
float Kp = 2.0;
float Ki = 0.5;
float Kd = 1.0;
float errorea = 0;
float azkenErrorea = 0;
float integrala = 0;

float kalkulatuPID(float helburua, float sarrera) {
  errorea = helburua - sarrera;
  integrala += errorea;
  float deribatua = errorea - azkenErrorea;
  float irteera = Kp * errorea + Ki * integrala + Kd * deribatua;
  azkenErrorea = errorea;
  return irteera;
}

Webgunearen Sorrera

Webgune hau diseinatu eta garatu genuen gure proiektua dokumentatzeko, HTML5, CSS3 eta JavaScript erabiliz esperientzia interaktiboa sortzeko.

Webguneak proiektuaren alderdi bakoitzerako atalak, irudi galeriak, erakustaldi bideoak eta dokumentazio teknikoa barne hartzen ditu.

Eduki guztia gure taldeak garatu zuen, txantiloi edo sortzaile automatikoen erabilera minimizatuz.

Gure proiekturako garrantzitsuak diren Laneko Segurtasun eta Osasun agenteak zuzen identifikatu genituen:

Langileen ordezkariak prebentzio gaietan

LSO eta LAP Funtzioak

Laneko Segurtasun eta Osasun eta Lan Arriskuen Prebentzio agenteen funtzioak modu egokian azaldu genituen:

  • Enpresaburua: Langileen segurtasuna eta osasuna bermatzea, arriskuak ebaluatzea, prebentzio jarduera planifikatzea, ekipamendu eta prestakuntza ematea.
  • Langileak: Ekipamendua zuzen erabiltzea, arrisku egoerak jakinaraztea, enpresaburuarekin lankidetzan aritzea prebentzio gaietan.
  • Prebentzio Ordezkariak: Zuzendaritzarekin lankidetzan aritzea, hobekuntzak sustatzea, enpresaburuak kontsultatzea, araudiaren betetzea zaintzea.
  • Segurtasun eta Osasun Batzordea: Prebentzio planen elaborazioan parte hartzea, arriskuen prebentzio metodoei buruzko ekimenak sustatzea.
  • Prebentzio Zerbitzuak: Enpresaburua, langileak eta ordezkariak aholkatzea eta laguntzea prebentzio gaietan.
