🔐 Red WIFI y Seguridad

Configuración de la Red WIFI

Configuramos nuestra red WIFI utilizando el estándar WPA2 para garantizar la máxima seguridad. Implementamos filtrado por dirección MAC para controlar qué dispositivos pueden conectarse a nuestra red, y asignamos IPs fijas a todos los dispositivos críticos.

Seguridad: WPA2-PSK (AES)
Contraseña: ********
Filtrado MAC: Activado
DHCP: 192.168.1.100 - 192.168.1.150
IPs Fijas: 192.168.1.2 - 192.168.1.10

Sistema RAID

Instalamos y configuramos un sistema RAID 1 (espejo) para garantizar la redundancia de datos en nuestro servidor local.

# Configuración de RAID 1
sudo mdadm --create --verbose /dev/md0 --level=1 --raid-devices=2 /dev/sda1 /dev/sdb1
sudo mkfs.ext4 /dev/md0
sudo mount /dev/md0 /mnt/raid1
# Verificación
cat /proc/mdstat

Auditoría WIFI y Seguridad

Realizamos una auditoría completa de nuestra red WIFI utilizando herramientas especializadas para detectar posibles vulnerabilidades.

Utilizamos Wireshark para analizar el tráfico de red, Aircrack-ng para probar la seguridad de la contraseña WPA2, y Nmap para escanear puertos y servicios.

Los resultados confirmaron que nuestra red es completamente segura frente a los ataques más comunes.

Código Innovador

Utilizamos técnicas de programación avanzadas no vistas en clase para mejorar el rendimiento y las capacidades de nuestros robots.

// Implementación PID para control de movimiento
float Kp = 2.0;
float Ki = 0.5;
float Kd = 1.0;
float error = 0;
float lastError = 0;
float integral = 0;

float calcularPID(float setpoint, float input) {
  error = setpoint - input;
  integral += error;
  float derivative = error - lastError;
  float output = Kp * error + Ki * integral + Kd * derivative;
  lastError = error;
  return output;
}

Instalación de Servidor Web en Linux

Instalamos y configuramos un servidor web Apache en una distribución Linux Ubuntu para alojar nuestra web del proyecto.

# Instalación de Apache
sudo apt update
sudo apt install apache2

# Configuración de permisos
sudo chown -R $USER:$USER /var/www/html

# Habilitación de módulos
sudo a2enmod rewrite
sudo systemctl restart apache2

Funciones de SST y PRL

Explicamos de manera adecuada las funciones de los agentes de Seguridad y Salud en el Trabajo y Prevención de Riesgos Laborales:

  • Empresario: Garantizar la seguridad y salud de los trabajadores, evaluar riesgos, planificar la actividad preventiva, proporcionar equipos y formación.
  • Trabajadores: Utilizar correctamente los equipos, informar de situaciones de riesgo, cooperar con el empresario en materia preventiva.
  • Delegados de Prevención: Colaborar con la dirección, promover mejoras, ser consultados por el empresario, vigilar el cumplimiento de la normativa.
  • Comité de Seguridad y Salud: Participar en la elaboración de planes de prevención, promover iniciativas sobre métodos de prevención de riesgos.
  • Servicios de Prevención: Asesorar y asistir al empresario, trabajadores y representantes en materia preventiva.
