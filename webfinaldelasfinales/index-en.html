WIFI Network Configuration

We configured our WIFI network using the WPA2 standard to ensure maximum security. We implemented MAC address filtering to control which devices can connect to our network, and assigned fixed IPs to all critical devices.

WIFI Audit and Security

We conducted a comprehensive audit of our WIFI network using specialized tools to detect potential vulnerabilities.

We used Wireshark to analyze network traffic, Aircrack-ng to test the security of the WPA2 password, and Nmap to scan ports and services.

The results confirmed that our network is completely secure against the most common attacks.

Innovative Code

We used advanced programming techniques not covered in class to improve the performance and capabilities of our robots.

// PID implementation for movement control
float Kp = 2.0;
float Ki = 0.5;
float Kd = 1.0;
float error = 0;
float lastError = 0;
float integral = 0;

float calculatePID(float setpoint, float input) {
  error = setpoint - input;
  integral += error;
  float derivative = error - lastError;
  float output = Kp * error + Ki * integral + Kd * derivative;
  lastError = error;
  return output;
}

OSH and ORP Functions

We adequately explained the functions of Occupational Safety and Health and Occupational Risk Prevention agents:

  • Employer: Ensure workers' safety and health, assess risks, plan preventive activity, provide equipment and training.
  • Workers: Correctly use equipment, report risk situations, cooperate with the employer in preventive matters.
  • Prevention Delegates: Collaborate with management, promote improvements, be consulted by the employer, monitor compliance with regulations.
  • Safety and Health Committee: Participate in the development of prevention plans, promote initiatives on risk prevention methods.
  • Prevention Services: Advise and assist the employer, workers, and representatives in preventive matters.
